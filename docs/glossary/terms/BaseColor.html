Base Color

Tags: PBR, Materials, Texturing, Unreal Engine

Definition Overview

Base Color is the pure color information of a material in a Physically Based Rendering (PBR) workflow. It represents the inherent color of a surface without lighting, shadows, highlights, or reflections.

In Unreal Engine, the Base Color texture is connected to the Base Color input of a Material and is combined with Roughness, Metallic, and Normal data at runtime to produce the final shaded result.

Critical Student Concept: Valid Base Color Values

One of the most common mistakes students make is creating Base Color textures that are too dark. In the real world, almost no materials have Base Color values close to pure black.

IMPORTANT:
Base Color values must be physically believable. If a material only looks correct under extremely bright lighting, the Base Color texture is likely incorrect.

Shading, shadows, and contrast are created by lighting and surface response — not by darkening the Base Color map.
